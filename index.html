记录人生足迹

追求人生足迹是个永无止境的过程. ———-题记

每个人从呱呱落地来到这个世界，就不断地在自己人生里留下深刻的足迹，时间好比一个天平，可以衡量他们的成果的重量；时间犹如一本史书，可以记录他们的人生留下的篇章；时间仿佛是一条鞭子，催促他们加快人生的脚步，我也不例外。
昔日的我，是一个不因碌碌无为而羞耻的孩子，整日游手好闲，无心向学，结果在人生的第一个转折点就踏上了一条崎岖的弯路，给人生留下了难忘的足迹。
13岁的我，来到一所很普通的中学念书，在这里转眼间就度过了三个春秋，面临着中考，面对着前途的方向，我虽然幸运地通过了，但是人生的这条崎岖路仍没有见到平坦的地方，这令我有点点后悔之前所走的足迹。
现在，我是一名普通学校的高三学生。面对人才的优势，我害怕自己技不如人；面对父母的期望，我害怕名落孙山；面对时间的逼切，我害怕“恐年岁之不吾已”……. 但是，看见妈妈白发苍苍，我又怎么忍心放弃呢？
我深深地感受到妈妈的期望，感动着，同时也后悔着，自责着，百般滋味在心头。妈妈，是我不孝，是我不乖，你为了我的学费奔波劳累，彻夜不眠。我无以为报，只能在这次高考用我优异的成绩来补偿我以往的过错。 高三这些日子，是我人生中第一次感受到命运是真正地掌握在自己手中，人生每一个足迹，影响你的未来的前途。即使我的人生是多么坎坷，多么崎岖，我也要走下去，为我的人生留下美丽的足迹。

我的足迹将是…….. 你呢？

hexo常用命令笔记

hexo

npm install hexo -g #安装
npm update hexo -g #升级
hexo init #初始化

简写

hexo n “我的博客” == hexo new “我的博客” #新建文章

hexo p == hexo publish

hexo g == hexo generate#生成

hexo s == hexo server #启动服务预览

hexo d == hexo deploy#部署
